# Project Memory

## Core
Shinely — cleaning jobs marketplace. Primary #A855F7, lavender palette. Plus Jakarta Sans font.
Mobile-first, Uber-style. Bottom tabs: Jobs, Schedules, Chat, Premium, Profile.
Lovable Cloud backend with profiles, jobs, job_applications, schedules, conversations, messages, subscriptions, team_members, team_invites.
3 tiers: Free (2 jobs/day, 10% fee, 1 schedule), Premium $9.99 (3 jobs/day, 10% fee, 2 schedules), Pro $19.99 (unlimited, 5% fee).
Worker types: Cleaner (has transport) and Helper Partner (no transport). Set during signup.
