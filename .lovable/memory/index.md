# Project Memory

## Core
Shinely — cleaning jobs marketplace. Primary #A855F7, lavender palette. Plus Jakarta Sans font.
Mobile-first, Uber-style. Bottom tabs: Jobs, Schedules, Chat, Premium, Profile.
Lovable Cloud backend with profiles, jobs, job_applications, schedules, conversations, messages, subscriptions.
Free cleaners: 2 jobs/day, 1 schedule contact. Premium: unlimited, $19.99/mo after 7-day trial.
Roles: cleaner, owner, admin. Admin hidden at /admin-login and /admin.
i18n: EN/PT/ES with LanguageProvider, language column on profiles.
Google OAuth via Lovable Cloud managed auth.
