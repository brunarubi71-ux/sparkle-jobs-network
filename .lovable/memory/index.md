# Project Memory

## Core
Shinely — cleaning jobs marketplace. Primary #A855F7, lavender palette. Plus Jakarta Sans font.
Mobile-first, Uber-style. Bottom tabs: Jobs, My Jobs, Earnings, Chat, Premium, Profile (cleaner) / Post Job, My Jobs, Sell, Chat, Profile (owner).
Lovable Cloud backend with profiles, jobs, job_applications, schedules, conversations, messages, subscriptions, rewards.
Plans: Free (2 jobs/day, 1 contact, 10% fee), Pro $9.99 (DB: "pro", 3 jobs/day, 2 contacts, 10% fee), Premium $19.99 (DB: "premium", unlimited, 5% fee). 7-day trial.
Badge system: auto-assigned based on jobs completed, ratings, earnings. Synced on profile load.
