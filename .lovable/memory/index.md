# Project Memory

## Core
Shinely — cleaning jobs marketplace. Primary #A855F7, lavender palette. Plus Jakarta Sans font.
Mobile-first, Uber-style. Role-based bottom tabs: Cleaner vs Owner.
Lovable Cloud backend with profiles, jobs, job_applications, schedules, conversations, messages, subscriptions, reviews, portfolio_photos, rewards.
Free cleaners: 2 jobs/day, 1 schedule contact. Premium: unlimited, $19.99/mo after 7-day trial.
10% platform commission on all jobs. Anti-bypass chat filtering blocks contact sharing before payment.
Job flow: Open → Applied → Hired → In Progress → Completed. Both parties can review after completion.
